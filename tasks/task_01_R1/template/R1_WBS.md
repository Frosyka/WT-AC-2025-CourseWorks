- Эпик A. Модель данных и миграции

- Эпик B. API и интеграция данных

	- Задачи: B1. Реализовать endpoints CRUD; B2. Реализовать приём Reading; B3. Добавить пагинацию и фильтры

- Эпик D. UI / графики

- Эпик F. Окружение и CI/CD
